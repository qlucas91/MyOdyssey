# MyOdyssey

Scientific Innovation Dashboard for PEF, Alternative Proteins, and Bioactivity Research.

## Quick Start (Streamlit Cloud)

1. Push this repository to GitHub (private recommended).
2. Go to [share.streamlit.io](https://share.streamlit.io/) and connect your GitHub.
3. Select this repository, branch `main`, main file `app.py`.
4. In Settings > Secrets, add: `OPENAI_API_KEY = "sk-your-key-here"`
5. Click Deploy.

## Features

- **Weekly Top 10:** Crossref + OpenAlex dual-engine search with relevance scoring.
- **Idea Lab:** AI-powered research idea generation.
- **5-Year Retrospective:** High-impact papers from 2020-2025.
- **Researcher Radar:** Global leaders in PEF and alternative proteins.
- **Raw Materials & Chat:** Interactive metabolite database with search.

## Automation

GitHub Actions updates data every Monday and Friday at 10:00 AM CET.
